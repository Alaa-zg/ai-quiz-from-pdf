# 🧠 AI-Powered Interactive Quiz Generator

## 📋 Overview
This project transforms educational PDF content into a fully interactive quiz web app using AI and front-end technologies. It helps reinforce learning with instant feedback and smart progress tracking.

## ⚙️ Features
- 15+ quiz questions auto-generated from PDF content
- Mix of multiple-choice, fill-in-the-blank, and scenario questions
- Correct answers shown for wrong responses with detailed explanation & example
- Manual “Next” button to control quiz flow
- Mobile-responsive, animated interface
- Final performance summary and suggestions

## 🧠 How It Works
1. Educational PDF content analyzed by ChatGPT-4o
2. HTML, CSS, and JavaScript code generated automatically
3. Code copied into VS Code to launch the app
